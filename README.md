# Web of Mephala - Entrance Patch (OpenMW)

Forces you to teleport in the correct location when entering the new fancy Morag Tong HQ.

Brute force solution to load order issue or a mod conflict. I'm not sure myself why exactly it happens, but I figured it would be easier to patch it with quick and dirty lua script than debug my mod conflicts even considering everything should just work.

Obviously requires [Web of Mephala Morag Tong Headquarters](https://www.nexusmods.com/morrowind/mods/55163). Load order doesn't matter.
